# Python 學習平台 - 前端界面

這是一個基於 React 的互動式 Python 學習平台前端界面。

## 功能特點

### 📚 三種學習模式
1. **講解模式**：詳細的知識點講解，支持代碼高亮顯示
2. **測驗模式**：選擇題測試，即時顯示答案和解析
3. **練習模式**：情景問題，需要動手編寫代碼

### 🎨 界面特色
- ✅ **可折疊側邊欄**：點擊按鈕隱藏/展開課程列表
- ✅ **可調整分欄**：拖動邊界調整三個面板的寬度
- ✅ **Monaco Editor**：VS Code 同款編輯器，支持語法高亮和自動補全
- ✅ **即時輸出**：運行代碼後立即顯示結果
- ✅ **響應式設計**：美觀的漸變色主題和流暢動畫

### 📖 學習內容
已內建由淺入深的 Python 課程：
- 第一章：Python 基礎
  - 什麼是 Python？
  - 變數與數據類型
- 第二章：運算與表達式
  - 算術運算
- 第三章：條件判斷
  - if 語句

## 安裝與運行

### 1. 安裝依賴

\`\`\`bash
cd frontend
npm install
\`\`\`

### 2. 啟動開發服務器

\`\`\`bash
npm start
\`\`\`

應用將在 http://localhost:3000 打開

### 3. 構建生產版本

\`\`\`bash
npm run build
\`\`\`

## 使用說明

### 學習流程
1. 從左側邊欄選擇一個課程
2. 閱讀講解內容
3. 點擊「測驗」按鈕進行選擇題測試
4. 答題後查看解析
5. 點擊「練習」按鈕進入編程練習
6. 在中間的編輯器中編寫代碼
7. 點擊「運行代碼」查看右側的輸出結果

### 調整面板大小
- 將鼠標移到面板之間的分隔線上
- 按住並拖動可調整面板寬度
- 支持三欄自由調整

### 側邊欄操作
- 點擊右上角的 ✕ 按鈕隱藏側邊欄
- 點擊左上角的 ☰ 按鈕展開側邊欄
- 點擊章節標題展開/收起子章節

## 技術棧

- **React 18**：前端框架
- **Monaco Editor**：代碼編輯器（VS Code 核心）
- **React Markdown**：Markdown 渲染
- **React Syntax Highlighter**：代碼語法高亮

## 注意事項

⚠️ **這是前端演示版本**
- 目前代碼執行是前端模擬，僅支持簡單的 print() 語句
- 要實現完整的 Python 代碼執行，需要連接後端 API
- 建議後端使用 Flask/Django 配合 Python 解釋器
