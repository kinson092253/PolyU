# 视频制作工具推荐和完整流程

## 🎬 推荐工具组合（免费 + 专业）

### 方案 A: 完全免费方案 ⭐⭐⭐⭐⭐
**适合：预算有限，追求性价比**

1. **录屏**: OBS Studio (免费开源)
   - 下载：https://obsproject.com/
   - 优点：专业级录屏，支持场景切换
   - 设置：1080p, 30fps

2. **AI 配音**: ElevenLabs (免费 10 分钟/月)
   - 网址：https://elevenlabs.io/
   - 优点：最自然的 AI 语音，支持多语言
   - 备选：Azure TTS (免费 5M 字符/月)

3. **视频编辑**: DaVinci Resolve (免费版功能强大)
   - 下载：https://www.blackmagicdesign.com/products/davinciresolve
   - 优点：专业级编辑，色彩校正，字幕
   - 替代：Shotcut (更轻量)

4. **PPT/图片制作**: Canva (免费版)
   - 网址：https://www.canva.com/
   - 用途：制作标题卡、概念图

## 📝 完整制作流程

### 第 1 步：准备 AI 配音
```bash
# 1. 打开 ElevenLabs
# 2. 选择语音：推荐 "Adam" (男声) 或 "Bella" (女声)
# 3. 粘贴脚本的【配音】部分
# 4. 设置：Stability: 50%, Clarity: 75%, Style: 0%
# 5. 生成并下载 MP3 文件
# 6. 命名：1_1_scene_1.mp3, 1_1_scene_2.mp3 等
```

**批量生成技巧**：
- 为每个场景分别生成音频
- 方便后期调整和重录

---

### 第 2 步：准备视觉素材
```bash
# 讲解部分（PPT 或图片）：
1. 打开 Canva
2. 选择 "Presentation" 模板 (1920x1080)
3. 制作：
   - 标题卡（课程名称）
   - 概念说明图（关键词 + 图标）
   - 总结卡
4. 导出为 PNG 或 PDF

# 导出设置：
- 分辨率：1920x1080 (Full HD)
- 格式：PNG (高质量)
```

---

### 第 3 步：录制代码演示
```bash
# OBS Studio 设置：
1. 打开 OBS
2. 添加来源 → 显示器捕获（选择主显示器）
3. 设置 → 输出：
   - 录制格式：MP4
   - 编码器：x264
   - 比特率：8000 Kbps
   - 分辨率：1920x1080
   - FPS：30

4. 开始录制前：
   - 打开你的 Python 平台（localhost:3000）
   - 最大化浏览器窗口
   - 隐藏不必要的标签页
   - 准备好要打的代码（放在旁边参考）

5. 录制技巧：
   - 打字速度：中等偏慢
   - 每行代码打完暂停 1 秒
   - 运行前暂停 1 秒（像在检查代码）
   - 结果显示后暂停 2-3 秒
   - 如果打错了，Backspace 删除再重打（更真实）

6. 停止录制：
   - 文件自动保存到"视频"文件夹
   - 重命名：1_1_demo_1.mp4
```

**录制清单**：
- ✅ 1.1: Hello World 演示
- ✅ 1.1: 多行 print 演示
- ✅ 1.2: 创建变量演示
- ✅ 1.2: 变量运算演示
- ✅ 1.2: 修改变量演示
- ✅ 1.3: print 高级用法演示
- ✅ 1.3: input 基础演示
- ✅ 1.3: 完整交互程序演示

---

### 第 4 步：视频编辑（DaVinci Resolve）
```bash
# 1. 创建项目
打开 DaVinci Resolve → 新建项目 → "Chapter_1_Lessons"

# 2. 导入素材
File → Import Media
导入：
- 所有 MP3 音频文件
- 所有 PNG 图片
- 所有录屏 MP4 文件

# 3. 创建时间线
右键 → 创建新时间线 → "Lesson_1_1"
设置：1920x1080, 30fps

# 4. 编辑 Lesson 1.1
Timeline 操作：
a) 拖入场景 1 图片（标题卡）
b) 拖入对应音频到音轨
c) 调整图片长度匹配音频
d) 拖入场景 2 图片
e) 拖入场景 2 音频
...
f) 拖入录屏视频
g) 拖入操作讲解音频
h) 同步音频和视频

# 5. 添加转场
在片段之间右键 → Add Transition → Cross Dissolve (0.5秒)

# 6. 添加字幕（可选）
Fairlight 页面 → Transcribe Audio → 自动生成字幕
或手动添加：Edit → Captions → New Caption Track

# 7. 音频调整
Fairlight 页面：
- 调整音量：配音 -3dB，背景音乐 -20dB
- 降噪：Effects → Noise Reduction

# 8. 导出视频
Deliver 页面：
- Format: MP4
- Codec: H.264
- Resolution: 1920x1080
- Quality: High
- 文件名：Lesson_1_1_What_is_Python.mp4
```

## ⚙️ OBS 快速设置指南

### 初次安装配置
```bash
1. 运行 OBS → 自动配置向导
2. 用途：选择"仅用于录制"
3. 分辨率：1920x1080
4. FPS：30

5. 手动调整：
设置 → 视频
- 基础分辨率：1920x1080
- 输出分辨率：1920x1080
- FPS：30

设置 → 输出 → 录制
- 录制路径：选择你的项目文件夹
- 录制格式：MP4
- 编码器：Software (x264)
- 比特率：8000 Kbps
```

### 场景设置
```bash
1. 来源 → + → 显示器捕获
2. 命名："屏幕录制"
3. 选择你的主显示器
4. OK

可选：
- 添加"音频输入捕获"（如果你要录解说）
- 添加"图像"（如果要加水印）
```

## 🎤 ElevenLabs 配音技巧

### 最佳语音选择
- **男声**: Adam (清晰专业) 或 Josh (友好温暖)
- **女声**: Bella (清晰标准) 或 Elli (活泼年轻)

### 配音参数设置
```
Stability: 50%     (太高会呆板，太低会不稳定)
Clarity: 75%       (高清晰度适合教学)
Style Exaggeration: 0%  (教学视频保持中性)
```

### 优化技巧
1. **断句**：用句号和逗号帮助 AI 理解节奏
2. **数字**：用文字写出 "twenty-five" 而不是 "25"
3. **代码**：用空格隔开，如 "print open parenthesis"
4. **重新生成**：不满意就重新生成，直到满意

## 📊 视频规格建议

### 标准规格
```
分辨率：1920x1080 (Full HD)
帧率：30 fps
比特率：8000 Kbps (视频) + 192 Kbps (音频)
格式：MP4 (H.264)
音频：AAC, 48kHz
```

### 文件大小估算
- 3 分钟视频 ≈ 180 MB
- 5 分钟视频 ≈ 300 MB
- 10 分钟视频 ≈ 600 MB
