## [场景 2: Python 简介] (30 秒)
**画面**: Python logo + 关键词动画
**配音**:
Python is a high-level, interpreted programming language created by Guido van Rossum in 1991. It's designed to be easy to read and write, with a syntax that emphasizes code readability. Python is used everywhere - from web development and data science, to artificial intelligence and automation. Major companies like Google, Netflix, and NASA use Python in their daily operations.

---

## [场景 3: 为什么学 Python] (30 秒)
**画面**: 特点列表动画
**配音**:
Why should you learn Python? First, it has a simple and clean syntax that's perfect for beginners. Second, it has a massive community and extensive libraries for almost any task. Third, it's versatile - you can build websites, analyze data, create games, and even program robots. And finally, Python developers are in high demand with competitive salaries.

---

## [场景 4: 实际操作 - 第一个程序] (90 秒)
**画面**: 切换到代码编辑器录屏
**配音 - 操作讲解**:
Now, let's write our very first Python program. I'm going to open the code editor here. 

(开始打字)
We'll start with the classic "Hello, World!" program. Type: print, open parenthesis, single quote, Hello comma space World exclamation mark, close quote, close parenthesis.

(pause 1 秒，代码显示完整)
Let me explain what this does. The print function tells Python to display text on the screen. The text we want to display goes inside the parentheses, enclosed in quotes. These quotes tell Python that this is a text string, not code.

(点击运行按钮)
Now when we click Run, watch what happens.

(显示输出)
Perfect! You can see "Hello, World!" appears in the output panel below. Congratulations! You've just written and executed your first Python program!

**操作时的代码**:
```python
print('Hello, World!')
```

---

## [场景 5: 扩展示例] (40 秒)
**画面**: 继续录屏操作
**配音 - 操作讲解**:
Let's try something more. Let me clear this and write a new line.

(打字: print('Welcome to Python!'))
Here I'm printing a different message: Welcome to Python.

(再打一行: print('Python is awesome!'))
And another line: Python is awesome.

(点击运行)
Now when we run this, both messages appear, one after the other. Python executes your code from top to bottom, line by line.

**操作时的代码**:
```python
print('Welcome to Python!')
print('Python is awesome!')
```
